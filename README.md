# DemocraTICon 2026 — 🥉 3. místo v národní soutěži (showcase)

Kompletní produkce kolem národní soutěže o demokracii **DemocraTICon 2026**: AI-generované vzdělávací epizody, mikrokurz [Elbot](https://hphugosm.github.io/tym-trampolin-web/projekty/elbot-microlearning/) a záznamy ze soutěže AI-sestříhané s titulky pro [Instagram týmu](https://www.instagram.com/tym_trampolin/). Tento repozitář je **showcase výsledků**.

## Cíl

Vytvořit srozumitelné vzdělávací mikro-epizody („Farma růží" — komunitní rozhodování, „Mezinárodní právo" — úřední proces) kompletně AI pipeline: od postav přes scény až po hotové klipy.

## Jak vznikalo (vysoká úroveň)

1. AI generace postav a prostředí (iterace, dokud scéna nesedí)
2. Image→video generace jednotlivých záběrů
3. Automatické zpracování: odstranění ticha → spojení klipů → exporty pro YouTube i vertikální formáty

## Použité nástroje

AI obrazová a video generace · Python + FFmpeg pipeline
